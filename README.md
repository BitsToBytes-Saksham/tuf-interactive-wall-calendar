# Interactive Wall Calendar

A polished, interactive wall calendar component built with Next.js and Tailwind CSS.

## Features
- Wall calendar aesthetic with spiral binding rings and monthly hero images
- Date range selection with live hover preview
- Indian public holidays marked with tooltips
- Per-month notes with localStorage persistence
- Season labels based on current month
- Fully responsive — desktop side-by-side, mobile stacked

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack
- Next.js 14
- Tailwind CSS
- React Hooks only (useState, useEffect, useCallback, useMemo)
